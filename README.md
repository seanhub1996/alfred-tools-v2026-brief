## Overview

Alfred Tools is a research-oriented toolkit made for Claude Code-driven analysis. Given a company name or ticker, it assembles a fully cited brief that brings together company background, trading comps, recent news, earnings notes, and referenced facts in a single workflow.

It is built for fast-moving research tasks that still need a clean structure. In addition to the main brief, the project can produce a web view and a print-friendly PDF, and it keeps completed briefs in a searchable research library for future comparison and review.

## What it does

- Converts a company name or ticker into a structured research brief
- Retrieves live trading comps, news, earnings takeaways, and sourced facts
- Outputs both a web page and a print-ready PDF
- Stores briefs in a searchable research library
- Runs as a Claude Code plugin
- Supports a cloned repository workflow for local execution
- Focused on company research and analyst-style coverage
- Built for repeatable brief creation

## FAQ

**How do I refresh the project?**  
Pull the newest changes from the repository and run the brief generation flow again in Claude Code.

**Where are the outputs saved?**  
The project can create a web page, a PDF, and saved entries in the research library.

**Can I use different inputs?**  
Yes. The workflow accepts both company names and tickers.

**What should I check if a brief seems incomplete?**  
Run the generation flow again and confirm the source data, network access, and input format are correct.

**Is this only meant for a single run?**  
No. The searchable research library is intended for repeated research sessions and comparison across names or tickers.
